# spring cloud

## 版本说明

| 版本号        | 版本说明  | 版本用途                |
|------------|-------|---------------------|
| BUILD      | 开发版   | 开发团队内部使用            |
| GA         | 稳定版   | 基本上可以用了，还有些bug      |
| PRE(M1,M2) | 里程碑版  | GA修复版               |
| RC         | 候选发布版 | 发行前观察期，高级bug修复      |
| SR         | 正式发布版 | SR1、SR2等等，修复大bug或优化 |
