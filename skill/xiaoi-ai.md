## 第一天

```tex
1、以下哪些属于传统客服面临的问题？
答：服务效率难以有效提升、人力成本持续居高不下、服务质量缺乏管理手段

答：上海科委
3、以下哪些是智能客服建设的关键步骤？
答：系统定位、团队支持、建设规划
4、什么技术被称为是AI皇冠上的明珠？
答：自然语言处理
5、NLP包括哪几部分？ 
答：NLP包括NLU（自然语言理解）和NLG（自然语言生成）两部分。
6、下列哪些是NLP的主要应用方向？ 
答：自动问答、信息检索、推荐系统、情感分析等
7、open平台中可以新增哪类知识？ 
答：文字、图片、音频、文字菜单、图文、视频
8、open平台的知识辞典中，有哪些类型？ 
答：同义词和同类词     
9、下列哪谢属于语音技术？ 
答：语音识别、声纹识别、语音合成
10、下列哪些属于机器视觉的应用？ 
答：多目标跟踪、、长文本检测、人体姿态估计、物体识别、人脸识别、语义分割
11、谁是AI之父？ 
答：马文闵斯基
12、AI发展史上里程碑的一次事件是什么？ 
答：达特茅斯会议
```

## 第二天

```tex
1、实体机器人分哪几类？
答：工业机器人、商用服务机器人、家庭服务机器人 
2、实体机器人可以应用到银行的哪些业务场景？
答：营销、服务、运营、管理
3、实体机器人的应用价值有哪些？
答：能力批处理、工作流程化、人员易管理、服务不间断、营销不设防
4、知识融合平台的应用场景有哪些？
答：客服机器人、智能在线辅助坐席系统、智能电话在线辅助系统、内部员工应用
5、智能知识融合平台的要素有哪些？ 
答：基于AI平台的基础保障、依托底层多模型的企业数据仓库、采编流程化、企业级权限管理
6、构建智能知识融合平台的基本步骤有哪些？ 
答：平台规划、知识建设、应用推广、平台优化
7、智能外呼技术的优势有哪些？ 
答：现在主流的外呼主要是第二种，自动语音和人工结合的方式，成本高体验差。全智能交互式的外呼则能基于企业知识库的全局知识点，通过全语音交互方式更好的服务客户，技术较领先。
8、智能外呼的应用场景有哪些？ 
答：账务催收、用户回访、客户关怀、市场营销、信息通知、风控信审     
9、外呼项目成本一般包括哪些？ 
答：外呼系统、IVR系统带录音、语音识别系统、语音合成系统、语义+对话引擎、场景建设费用。
10、以催收业务为例，外呼系统的验收指标一般有哪些？ 
答：技术指标（识别准确率、响应时间）和业务指标（还款率、投诉率、接通率）
11、人工智能未来发展趋势有哪些？ 
答：感知智能向认知方向迈进、人工智能进入大规模商用阶段
12、组织如何应用AI进行智能化转型升级？ 
答：1）学习了解AI的基本概念和技术应用
    2）掌握如何从0开始建立AI应用的方法论
    3）组织及人才的准备
    4）确定应用场景，寻找解决方案
```

## 第三天

```tex
1、声音的特征有哪些？
答：音高、音强、音长、音色 
2、语音的特征有哪些？
答：由人发出、信息交流、语言特征
3、语音识别技术有哪几个步骤？
答：特征提取、声学比对、语言解码
4、容易被人工智能颠覆的行业的共同特点有哪些？ 
答：迫切的自动化、智能化需求；较好的数字化基础；成熟的业务流程；人力密集
5、哪些行业最容易被人工智能颠覆？ 
答：客户服务、金融、政府等行业
6、举例人工智能在政府领域的应用有哪些？ 
答：12345热线自流程派单、两会提案查重 、智慧党务、智慧法律服务亭等
7、构建智能服务系统的价值有哪些？ 
答：提升服务效率、优化服务体验、扩大服务能力。
8、AI应用落地的四要素有哪些？
答：Algorithm算法、Bigdata大数据、Computing算力、Domain 领域/场景 
9、连接革命到智能化革命是如何演化的？ 
答：1）信息化：低水平的数字化计算机化，信息孤岛，运营孤岛
    2）互联网化：资源(人/物—数据/信息)０边际成本的互联互通
    3）大数据化：数据 → 大数据，跨界可用数据资源的规模与形态
    4）智能化：可知，可达，可控，可预测
10、互联网化智能化的经济含义是什么？ 
答：交易：０边际成本；生产：０边际成本；边际革命：人类将逐步进入０边际成本时代；结果变成原因，逼迫转型升级
```

## 第四天

```tex
1、在智能化转型过程中，组织该如何变革？ 
答：从以企业/政府部门为中心转变为以用户/人民为中心
2、企业智能化转型的基础与准备有哪些？
答：1）管理的科学化、标准化、系统化、信息化程度；
    2）互联网化程度：业务体系形态，IT系统；
    3）数据化程度：数据互联互通的机制与系统，数据的可得性/可用性；
    4）知识化程度；
    5）企业经营管理目标与智能化目标定位；
    6）甲方智能化项目小组的组成。
3、CC智能技术的应用有哪些？
答：1）智能客服机器人：智能问答；
    2）智能外呼机器人：自动精准外呼营销；
    3）AI+BI：基于语料挖掘的分析与决策；
    4）客户画像、客户管理、客户跟踪；
    5）全流程处理与管控。
4、CC智能化的逐级演进与实现路径是怎样的？ 
答：1）对人工客服的赋能增效：传统手段+技术赋能
    2）智能知识库逐步积累与人工逐步替代：人机协作&人机替代
    3）访客端逻辑导向的服务运营体系的重构与优化
    4）分析决策工具与能力的提升
    5）机器人的人性化
5、为什么说cc部门是未来真正的核心部门？ 
答：1）“连接”中心：连接企业与客户的多渠道一体化运营
    2）“感知”中心：组织改善客户体验和运营体系的发起点
    3）“前指”中心：组织日常运营的中枢
    4）“参谋”中心：决策辅助
    5）范式改变了
6、外呼的目标是什么？ 
答：寻找客群进行外呼、表达用意、达成一致、创造价值
7、外呼项目的立项需要考虑什么问题？ 
答：技术是否成熟、是否有应用案例、投入产出比怎么计算、立项由谁发起。
8、智能外呼系统需要哪些软件？
答：PBX、外呼系统、IVR系统、智能交互系统、报表系统
9、如何选择智能语音外呼厂商？ 
答：技术匹配、经验丰富、支持服务到位、可运维、价格合理
10、VUI设计的原则有哪些？ 
答：1）以数据分析为设计基础
    2）高度拟人化的设计风格
    3）便于持续优化的设计框架
    4）以任务为导向的设计流程
    5）规范完备直观的设计产出
```

## 第六天

```tex
1、知识管理的发展历程是怎样的？ 
答：1）第一代: 文档电子化，实现局域网知识共享
    2）第二代:目录导航+关键字检索,提供文档知识查找能力
    3）第三代: 原子化知识库，实现知识点精准搜索
    4）第四代: 知识图谱，挖掘知识关系，提升智能性
哪些属于小i智能知识库知识模型？
答：结构化知识（faq模型）、半结构化知识（百科类、多维表模型）、非结构化知识（文档模型）
智能知识融合平台的评价指标有哪些？
答：业务覆盖率、业务准确率、知识服务能力、搜索准确率、翻找知识用时
4、知识融合平台基于AI平台的基础保障有哪些？ 
答：自然语言处理、语音/图像识别、机器学习等人工智能技术
5、知识融合平台的定位是什么？ 
答：知识融合平台是企业核心大脑，是企业智能化应用扩展的核心基础。
6、知识建设主要有哪些部分组成？
答：知识采编、知识审核、知识发布
7、在知识采编过程中，需要整理哪些素材？
答：显性知识（包括原知识库系统、其他业务系统、爬虫知识等）和隐形知识（包括个人经验知识、互动问答知识、服务过程知识、知乎等）
8、知识优化的三个类比分别是什么？
答：知识结构优化、知识内容优化、支撑能力优化
```

## 第七天

```tex
1、VCA中的Conversational AI 需要解决哪些问题？
答：日常聊天、基础问答、复杂场景下解决任务
2、Conversation AI的发展现状是如何的？
答：以自然语言处理为核心技术、结合深度学习加以辅助、配合会话管理等传统技术手段
3、智能客服机器人的未来展望是如何的？
答：职能扩展会从服务到营销；结构覆盖会从结构化到非结构化；模态会更丰富从文本到语音、情感、图像
4、智能客服应用价值是什么？
答：提升效率、节约成本、降低人工
5、智能客服机器人的目标定位包括哪些？
答：应用目标、服务对象、业务范围、品牌区格
6、智能客服体系全景图中分为哪几层？
答：接入层、AI+服务层、AI+支撑层、AI+运营层
7、智能运营的痛点有哪些？ 
答：从公司角度来看希望低成本高效率；从客户角度来看希望提升客户体验；从大环境角度看如何进行数据变现；从自身角度看如何体现自身价值。
8、智能运营的团队一般有哪些角色？ 
答：数据分析、AI训练、运营顾问、培训、统计汇报
9、如何完整的进行智能服务机器人的运营？
答：首先组建运营团队，通过分析用户数据，整理出系统直观的数据报表，并制定优化方案；再通过分析结果，持续AI训练，积累行业数据，持续优化体验；通过机器人深度语义理解能力，结合业务数据分析，定制个性化的多轮交互场景；并结合企业新业务需求，实现更多定制化功能的开发；最后新型人机结合的运营方式，离不开合理有效的工具支持。
```
